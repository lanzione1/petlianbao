# 客户管理页面重新设计方案

> 日期: 2026-02-25
> 目标: 重新设计客户管理前端页面，突出"人"而不是"宠物"

---

## 设计原则

1. **以客户（人）为中心** - 客户详情页优先显示客户信息
2. **减少操作层级** - 宠物档案内嵌在客户流程中，不需要单独跳转
3. **保持数据一致性** - 后端已支持多宠物，前端需适配

---

## 1. 客户列表页

### 搜索
- 搜索框：按客户姓名/手机号搜索
- 筛选标签：全部、流失客户

### 列表卡片显示
```
┌─────────────────────────────┐
│ 张三              138****0001 │
│ 2只宠物           ¥1,500    │
│ 最近: 2026-02-20  12次到店  │
└─────────────────────────────┘
```

- 主标题：客户姓名（去掉了宠物名）
- 副标题：宠物数量 + 手机号
- 统计：累计消费、到店次数

---

## 2. 客户详情页

### 页面结构
```
┌─────────────────────────────┐
│ 张三                        │
│ 138****0001                 │
├─────────────────────────────┤
│ 累计消费    到店次数  客单价 │
│ ¥1,500      12次     ¥125   │
├─────────────────────────────┤
│ 🐕 旺财（金毛）  🎂2月生日   │
│ 🐱 咪咪（英短）  💉3月疫苗   │
├─────────────────────────────┤
│ 消费记录                    │
│ 2026-02-20  洗澡+驱虫 ¥130 │
│ 2026-02-15  疫苗      ¥100 │
├─────────────────────────────┤
│ [新增消费]  [编辑客户]      │
└─────────────────────────────┘
```

### 宠物卡片
- 显示宠物名字、品种、物种图标
- 提醒标识：生日🎂 / 疫苗💉 / 驱虫🐛
- 点击可快速编辑宠物

---

## 3. 客户编辑页

### 页面结构
```
┌─────────────────────────────┐
│ 编辑客户                    │
├─────────────────────────────┤
│ 客户信息                    │
│ 姓名: [张三]               │
│ 手机: [13800000001]        │
│ 备注: [老客户]              │
├─────────────────────────────┤
│ 宠物档案 (2只)              │
│ ┌─────────────────────┐     │
│ │ 🐕 旺财  金毛       │编辑 │
│ └─────────────────────┘     │
│ ┌─────────────────────┐     │
│ │ 🐱 咪咪  英短       │编辑 │
│ └─────────────────────┘     │
│ [+ 添加宠物]                 │
└─────────────────────────────┘
```

### 宠物表单字段
- 名字 *、品种 *、物种（狗/猫/其他）
- 生日、体重
- 下次疫苗日期、下次驱虫日期
- 饮食习惯、过敏史、行为特点
- 备注

---

## 4. 交互流程

```
客户列表 → 点击客户 → 客户详情
                         ↓
                   宠物卡片点击 → 宠物编辑（弹窗或新页面）
                         ↓
                   编辑客户 → 客户编辑页
                         ↓
                   宠物管理区 → 添加/编辑/删除宠物
```

---

## 5. 需要修改的文件

### 小程序端
- `pages/customer/list.wxml/js` - 改为显示客户信息
- `pages/customer/detail.wxml/js` - 改为显示客户+宠物列表
- `pages/customer/add.wxml/js` - 改为客户编辑页（含宠物管理）
- 新增或复用 `pages/pet/edit` 作为宠物编辑

### API调整
- 客户列表API：增加宠物数量字段
- 客户详情API：返回客户+关联宠物列表
