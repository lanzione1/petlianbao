# 微信配置指南

## 概述

宠联宝需要配置以下微信能力：

| 功能 | 用途 | 配置位置 |
|------|------|----------|
| 小程序登录 | 商家端登录 | 微信公众平台 |
| 模板消息 | 预约提醒 | 微信公众平台 |
| 微信支付 | 收银收款 | 微信支付商户平台 |

---

## 一、注册微信小程序

### 1.1 注册地址

访问 [微信公众平台](https://mp.weixin.qq.com/) 注册小程序账号

### 1.2 选择类目

推荐类目：
- **商家自营** > **宠物服务**
- 或 **生活服务** > **其他生活服务**

> ⚠️ 需要营业执照

### 1.3 获取 AppID 和 AppSecret

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入 **开发管理** > **开发设置**
3. 复制 **AppID(小程序ID)** 和 **AppSecret(小程序密钥)**

```
AppID: wx1234567890abcdef
AppSecret: 1234567890abcdef1234567890abcdef
```

### 1.4 配置到项目

**后端配置** (`apps/backend/.env`):

```env
WECHAT_APP_ID=wx1234567890abcdef
WECHAT_APP_SECRET=1234567890abcdef1234567890abcdef
```

**小程序配置** (`apps/merchant-miniapp/project.config.json`):

```json
{
  "appid": "wx1234567890abcdef"
}
```

## 三、配置微信登录

### 3.1 登录流程

```
小程序                          后端                           微信服务器
  │                              │                                │
  │  1. wx.login()               │                                │
  │  ─────────────────────────>  │                                │
  │                              │                                │
  │  2. 返回 code                 │                                │
  │  <─────────────────────────  │                                │
  │                              │                                │
  │  3. POST /auth/wechat/login  │                                │
  │  ──────────────────────────────────────────────────────────>│
  │                              │                                │
  │                              │  4. code换取openid             │
  │                              │  ─────────────────────────────>│
  │                              │                                │
  │                              │  5. 返回 openid, session_key   │
  │                              │  <─────────────────────────────│
  │                              │                                │
  │  6. 返回 JWT token           │                                │
  │  <─────────────────────────  │                                │
```

### 3.2 后端实现 (已完成)

```typescript
// apps/backend/src/modules/auth/auth.service.ts

async wechatLogin(code: string) {
  // 1. 用 code 换取 openid
  const url = `https://api.weixin.qq.com/sns/jscode2session?appid=${appId}&secret=${appSecret}&js_code=${code}&grant_type=authorization_code`;
  
  const response = await axios.get(url);
  const { openid, session_key } = response.data;
  
  // 2. 查找或创建商家
  let merchant = await this.merchantsRepository.findOne({ where: { openid } });
  if (!merchant) {
    merchant = this.merchantsRepository.create({ openid, shopName: '新店铺' });
    await this.merchantsRepository.save(merchant);
  }
  
  // 3. 生成 JWT
  const token = this.jwtService.sign({ merchantId: merchant.id, openid });
  
  return { access_token: token, merchant };
}
```

## 四、配置模板消息（预约提醒）

### 4.1 申请模板

1. 登录微信公众平台
2. **功能** > **订阅消息**
3. 点击 **公共模板库**
4. 搜索并添加以下模板：

### 4.2 推荐模板

| 模板名称 | 用途 | 关键词 |
|----------|------|--------|
| 预约成功通知 | 客户预约成功 | 预约项目、预约时间、服务地址 |
| 预约提醒 | 预约前提醒 | 服务名称、预约时间、温馨提示 |
| 服务完成通知 | 服务完成后 | 服务项目、消费金额、服务时间 |

### 4.3 获取模板ID

添加模板后，复制 **模板ID**：

```
模板ID: xxxxxxxxxxxxxxxxxxxxxxxx
```

### 4.4 后端配置

```env

WECHAT_TEMPLATE_ID_APPOINTMENT=xxxxxxxxxxxxxxxxxxxx
WECHAT_TEMPLATE_ID_REMINDER=xxxxxxxxxxxxxxxxxxxx
WECHAT_TEMPLATE_ID_COMPLETE=xxxxxxxxxxxxxxxxxxxx
```

## 五、配置微信支付

### 5.1 注册商户号

1. 访问 [微信支付商户平台](https://pay.weixin.qq.com/)
2. 注册并完成企业认证
3. 绑定小程序 AppID

### 5.2 获取支付配置

**账户中心** > **API安全**：

| 配置项 | 说明 |
|--------|------|
| mch_id | 商户号 |
| api_key | API密钥 (32位) |
| api_v3_key | APIv3密钥 |
| serial_no | 证书序列号 |

### 5.3 下载证书

下载以下证书文件：
- `apiclient_key.pem` - 商户私钥
- `apiclient_cert.pem` - 商户证书

### 5.4 后端配置

```env
# apps/backend/.env

# 微信支付配置
WECHAT_PAY_MCH_ID=1234567890
WECHAT_PAY_API_KEY=your-32-character-api-key-here
WECHAT_PAY_API_V3_KEY=your-32-character-v3-key-here
WECHAT_PAY_SERIAL_NO=xxxxxxxxxxxxxxxxxxxx
WECHAT_PAY_NOTIFY_URL=https://api.petlianbao.com/api/v1/pay/notify
```

### 5.5 证书文件

将证书放到：
```
apps/backend/certs/
├── apiclient_key.pem
└── apiclient_cert.pem
```

### 5.6 安装支付SDK

```bash
cd apps/backend
pnpm add wechatpay-node-v3
```

### 5.7 支付接口示例

```typescript
import { Payment } from 'wechatpay-node-v3';

// apps/backend/src/modules/billing/wechat-pay.service.ts

@Injectable()
export class WechatPayService {
  private payment: Payment;

  constructor(private configService: ConfigService) {
    this.payment = new Payment({
      appid: configService.get('wechat.appId'),
      mchid: configService.get('wechat.pay.mchId'),
      serial_no: configService.get('wechat.pay.serialNo'),
      privateKey: fs.readFileSync('./certs/apiclient_key.pem'),
      apiv3_private_key: configService.get('wechat.pay.apiV3Key'),
    });
  }

  async createOrder(orderId: string, amount: number, openid: string) {
    const result = await this.payment.transactions_jsapi({
      description: '宠物服务',
      out_trade_no: orderId,
      amount: { total: amount * 100 }, // 单位：分
      payer: { openid },
      notify_url: this.configService.get('wechat.pay.notifyUrl'),
    });
    
    return result;
  }
}
```

## 六、获取 Access Token

### 6.1 全局唯一接口调用凭据

```typescript
// 后端缓存 access_token (有效期 2 小时)

async getAccessToken(): Promise<string> {
  // 检查缓存
  if (this.accessToken && Date.now() < this.tokenExpireAt) {
    return this.accessToken;
  }

  // 请求新 token
  const url = `https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=${appId}&secret=${appSecret}`;
  const response = await axios.get(url);
  
  this.accessToken = response.data.access_token;
  this.tokenExpireAt = Date.now() + (response.data.expires_in - 300) * 1000;
  
  return this.accessToken;
}
```

## 常见问题

### Q: 登录失败 "code been used"

A: `wx.login()` 获取的 code 只能使用一次，每次登录都需要重新调用

### Q: 模板消息发送失败

A: 检查：
1. 模板ID是否正确
2. 用户是否订阅了该模板
3. access_token 是否有效

### Q: 支付回调失败

A: 检查：
1. notify_url 是否可访问
2. 是否使用 HTTPS
3. 证书是否正确
