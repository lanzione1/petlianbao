## 二、功能模块详细设计

### 模块1: 商家管理 (已有基础，需增强)

#### 1.1 商家详情页增强
| 功能 | 说明 | 优先级 |
|------|------|--------|
| 基本信息展示 | 店铺名、联系方式、地址、营业时间 | ⭐⭐⭐ |
| 业务统计 | 客户数、预约数、交易额、客单价 | ⭐⭐⭐ |
| 趋势图表 | 近7天/30天交易趋势 | ⭐⭐ |
| 操作记录 | 审核记录、禁用记录 | ⭐⭐ |

#### 1.2 套餐管理 (新增)
| 功能 | 说明 | 优先级 |
|------|------|--------|
| 套餐配置 | 免费版/专业版功能对比 | ⭐⭐⭐ |
| 升级审核 | 专业版申请审核 | ⭐⭐⭐ |
| 到期提醒 | 15天/7天/3天到期提醒 | ⭐⭐ |
| 续费管理 | 续费记录、续费统计 | ⭐⭐ |

#### 1.3 商家状态流转
```
新注册 → 待审核 → 正常 → 禁用
           ↓        ↓
        拒绝     逾期
```

### 模块2: 数据分析 (新增)

#### 2.1 平台数据大盘
| 指标 | 说明 |
|------|------|
| 商家总数 | 总数/今日新增/本周新增 |
| 活跃商家 | 7天内有交易/预约的商家 |
| 客户总数 | 全平台累计客户数 |
| 交易总额 | 今日/本周/本月/累计 |
| 预约总数 | 今日/本周/本月 |
| 转化率 | 预约→到店→交易转化 |

#### 2.2 排行榜
| 类型 | 维度 |
|------|------|
| 商家排行 | 交易额/订单量/客户数 |
| 服务排行 | 热门服务TOP10 |
| 区域排行 | 按城市/区域统计 |

#### 2.3 趋势分析
- 日/周/月趋势图
- 同比/环比分析
- 增长预测
